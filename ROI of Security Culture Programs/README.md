# 📖 Project Overview

**ROI of Security Culture Programs** is a cybersecurity analytics project designed to measure and demonstrate the financial value of security awareness and security culture initiatives.

Organizations often invest heavily in:

* Security awareness training
* Phishing simulations
* Compliance programs
* Human risk management initiatives

However, security leaders frequently struggle to answer a critical question:

> **"What business value are we receiving from these investments?"**

This project provides a practical framework for quantifying the return on investment (ROI) of security culture programs using real-world financial analysis techniques, Python-based modeling, and data visualization.

# 💰 ROI Calculation Framework

The project applies standard financial analysis techniques.

### Monthly ROI

```text
ROI (%) =
(Net Benefits / Program Costs) × 100
```

Where:

```text
Net Benefits =
Total Benefits − Total Costs
```

---

### Cumulative ROI

```text
Cumulative ROI =
(Total Benefits − Total Costs)
/ Total Costs × 100
```

---

### Payback Period

The point at which:

```text
Cumulative Benefits >
Cumulative Costs
```

This indicates when the investment begins generating positive returns.
