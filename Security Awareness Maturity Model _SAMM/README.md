# 🛡️ Security Awareness Maturity Model (SAMM) Lab

## 📖 Overview

The **Security Awareness Maturity Model (SAMM) Lab** is a hands-on cybersecurity project designed to help learners assess, measure, and improve an organization's security awareness program using a structured maturity framework.

This lab guides students through the development of a complete **Security Awareness Maturity Assessment Platform**, including survey data collection, maturity scoring, statistical analysis, reporting, and visualization. Using Python and data analysis techniques, participants will learn how to evaluate organizational security culture and generate actionable recommendations for continuous improvement.

The project follows a maturity-based approach, enabling organizations to identify strengths, weaknesses, and opportunities for enhancing security awareness programs through measurable and repeatable assessment processes.

---

## 🎯 Learning Objectives

By completing this lab, students will learn how to:

* Implement the Security Awareness Maturity Model (SAMM) framework
* Build Python-based maturity assessment engines
* Generate and analyze security awareness survey data
* Calculate organizational maturity scores using weighted models
* Perform statistical analysis on awareness metrics
* Create visual reports and dashboards
* Generate executive-level recommendations
* Measure and improve organizational security culture
